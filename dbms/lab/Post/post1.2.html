Question 5
Consider the following set of relations
 
EMP ( emp_no,emp_name,dept_no,salary)
DEPT (dept_no,dept_name,location)
Write SQL query for the following
Find all the employees whose departments are located in 'Mumbai' and salary is greater than Rs. 30,000.
 
select emp_name from dept where dept_no and location='Mumbai';


select emp_name from emp where salary>30,000 and dept_no in (select dept_no from dept where location ='Mumbai');


select dept_no ,count(emp_no) from emp where salary > 30,000 group by dept_no;


select dept_no ,count(emp_no) from emp where salary > 30,000 group by dept_no;
Answer :select emp_name from emp where salary > 30,000 and dept_no in (select dept_no from dept where location 
= 'Mumbai');
