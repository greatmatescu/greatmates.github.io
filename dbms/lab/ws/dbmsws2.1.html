DBMS WORKSHEET 2.1

Question 1

Write a PL/SQL program to display the description against a grade according to the marks achieved by 
students in their final exams.

Code:
----to display grade according to marks
DECLARE
marks number(10);
BEGIN
marks:=79;
if(marks>=90)then
dbms_output.put_line('grade is A+');
elsif(marks>=80 and marks<=80)then
dbms_output.put_line('grade is A');
elsif(marks>=70 and marks<=80)then
dbms_output.put_line('grade is B+');
elsif(marks>=60 and marks<=70)then
dbms_output.put_line('grade is B');
elsif(marks>=50and marks<=60)then
dbms_output.put_line('grade is C+');
else
dbms_output.put_line('F fail');
end if;
END; 


Ques : 2

Write a PL/SQL program to check whether a given number is positive, negative
or zero.
code :

declare
n number := 16;
begin
if n <0 then
dbms_output.Put_line('Negative');
elsif n >0 then
dbms_output.Put_line('Positive');
else
dbms_output.Put_line('Zero');
end if;
end;
