                Question 2
                Which of the following is true about the following code snippet?
                DECLARE
                a number(3) := 100;
                BEGIN
                IF (a = 50 )
                THEN
                dbms_output.put_line('Value of a is 10' );
                ELSIF ( a = 75 )
                dbms_output.put_line('Value of a is 20' );
                ELSE
                dbms_output.put_line('None of the values is matching');
                END IF;
                dbms_output.put_line('Exact value of a is: '|| a );
                END; 
                 
                It has syntax error
                
                
                It will print 'None of the values is matching'.
                
                
                It will print
                None of the values is matching
                Exact value of a is: 100
                
                
                None of the above
                Answer It has syntax error

                Question 4
                Which of the following is not true about PL/SQL decision making structures?
                 
                The IF statement associates a condition with a sequence of statements enclosed by the keywords 
                THEN and END IF.
                
                
                The IF statement also adds the keyword ELSE followed by an alternative sequence of statement.
                
                
                The IF-THEN-ELSIF statement allows you to choose between several alternatives.
                
                
                PL/SQL does not have a CASE statement.
                Answer PL/SQL does not have a CASE statement.
