                Question 1
                
                To select only one copy of each set of duplicable rows which SQL SELECT command is used
                
                SELECT UNIQUE
                
                
                SELECT DIFFERENT
                
                
                SELECT DISTINCT 
                
                
                All of the above
                
                Answer:
                SELECT DISTINCT 
                
                Question 2
                
                Write a query to display employee details (Name, Department, Salary and Job) from EMP table.
                
                SELECT DISTINCT ename, deptno, sal, job FROM emp;
                
                
                SELECT ename, deptno, sal, job FROM emp;
                
                
                SELECT ename, deptno, sal FROM emp;
                
                
                SELECT * FROM emp;
                
                Answer: A
                SELECT ename, deptno, sal, job FROM emp;

                Question 4

                What will be theoutput of of the following SQL statement?
                
                SELECT person_id, Fname, lname FROM person;
                
                Show only columns (person_id, Fname, lname)
                
                Show only columns (person_id, Fname, lname) and rows related to these columns
                
                Show all columns except (person_id, Fname, lname)

                Show all rows
                
                Answer: B
                Show only columns (person_id, Fname, lname) and rows related to these columns
