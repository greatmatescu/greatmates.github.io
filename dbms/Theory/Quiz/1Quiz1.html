DBMS Quiz 1 


Que. A attribute in a relation is a foreign key if which of the following key from one relation is used as an attribute in
that relation .

Candidate Key

Super Key

Sub Key

Primary Key
Answer : D.Primary Key 

Que. Consider attributes ID , CITY and NAME . Which one of this can be considered as a super key ?

ID

CITY

NAME

CITY,ID
Answer :A. ID

Que. Which of the following is not involved in DBMS?

End Users

Data

Application Request

HTML
Answer :D. HTML

Que. Database is called as a collection of what?
Hide answer choices 
Module

Data

Programs

Instructions
Answer : B.Data

Que. Mapping of concepts and their relationships is represented by what?

Logical Schema

Physical Schema

Conceptual Schema

None of these
Answer : C.Conceptual Schema

Que. The key selected from the sets of candidate keys by database designer is called

Primary key

Secondary Key

Foreign key

Super key
Answer : A. Primary Key

Que. To remove a relation from an SQL database, we use which command from the following?

DELETE

DROP TABLE

REMOVE

PURGE
Answer : B. Drop Table

Que. The degree of the relationship is

number of Tables in a relationship

number of entities in a relationship

number of Row & Columns in a relationship

number of participating entities in a relationship
Answer : D. number of participating entities in a relationship
