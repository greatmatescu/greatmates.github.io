DBMS Quiz 2 


                Question 1
                Entity Relationship model consists of collection of basic objects called _________ and relationship among 
                these objects.
                Functions
                
                
                Models
                
                
                Entities
                
                
                None of these
                Answer : Entities 

                 
                Question 2
                What Relational algebra is :
                Data Definition Language
                
                
                Meta Language
                
                
                Procedural query language
                
                
                Non procedural language
                Answer : Procedural query language

                 
                Question 3
                In the three-tier architecture, the intermediate layer between database and client servers is classified as 
                functional server
                
                
                transaction server
                
                
                application server
                
                
                disk server
                Answer : application server

                 
                Question 4
                Subschema can be used to 
                create very different, personalized views of the same data
                
                
                present information in different formats
                
                
                hide sensitive information by omitting fields from the sub-schema's description
                
                
                All of the above
                Answer : All of the above

                 
                Question 5
                Which command removes all rows from a table without logging the individual row deletions. 
                Delete
                
                
                Drop
                
                
                Truncate
                
                
                Remove
                Answer : Truncate

                 
                Question 6
                The process of suppression of details regarding storage and data type to highlight important features for 
                better understanding is called 
                data abstraction
                
                
                structural abstraction
                
                
                client abstraction
                
                
                server abstraction
                Answer : data abstraction

                 
                Question 7
                In unary relational operations, the SELECT operation is partition of relation usually classified as 
                horizontal partition
                
                
                horizontal partition
                
                
                insert partition
                
                
                delete partition
                Answer : horizontal partition

                 
                Question 8
                Consider two tuples X and Y, the operation whose result includes tuples that are only in X but not in Y 
                is classified as 
                square of relation X and Y
                
                
                union of relation X and Y
                
                
                intersection of relation X and Y
                
                
                difference of relation X and Y
                Answer : difference of relation X and Y
